# 客户建档与账号管理功能增强

> **创建日期：** 2026-06-04
> **项目名称：** 工作统计系统 - 功能增强
> **状态：** 已确认

---

## 1. 客户/生态建档模块增强

### 1.1 档案列表筛选

**功能：**
- 支持按类型筛选：客户 / 生态伙伴
- 支持按行业筛选：文旅 / 住建 / 传媒 / 体育

**交互：**
- 筛选条件显示在表格上方
- 多条件可组合使用
- 清空筛选按钮

### 1.2 档案编辑

**功能：**
- 支持编辑档案的：名称、行业、类型、主营业务

**交互：**
- 列表每行有"编辑"按钮
- 点击后弹出模态框
- 表单预填充当前数据
- 保存后更新列表

### 1.3 档案删除

**功能：**
- 支持删除档案记录

**交互：**
- 列表每行有"删除"按钮
- 点击后弹出确认框，显示档案名称
- 需用户点击确认才执行删除

### 1.4 导出 Excel

**功能：**
- 将档案列表导出为 Excel 表格

**交互：**
- 按钮位于表格上方
- 点击后下载 `.xlsx` 文件
- 包含字段：名称、类型、行业、主营业务

## 2. 账号管理模块增强

### 2.1 用户列表状态切换

**功能：**
- 组长可切换用户状态：正常 / 停用

**交互：**
- 使用滑动按钮控件
- 滑动即触发状态更新（直接保存）
- 颜色标识：绿色=正常，红色=停用

### 2.2 用户角色编辑

**功能：**
- 组长可修改用户的角色：经理 / 组长

**交互：**
- 列表每行有"编辑"按钮
- 点击后弹出模态框
- 下拉选择角色
- 保存后更新列表

### 2.3 删除停用用户

**功能：**
- 组长可删除"停用"状态的用户
- "正常"状态用户不可删除

**交互：**
- 仅"停用"用户显示"删除"按钮
- 点击后弹出确认框，显示用户姓名和邮箱
- 需用户点击确认才执行删除

### 2.4 用户列表筛选

**功能：**
- 支持按角色筛选：经理 / 组长
- 支持按状态筛选：正常 / 停用

**交互：**
- 筛选条件显示在表格上方
- 多条件可组合使用
- 清空筛选按钮

### 2.5 导出 Excel

**功能：**
- 将用户列表导出为 Excel 表格

**交互：**
- 按钮位于表格上方
- 点击后下载 `.xlsx` 文件
- 包含字段：姓名、邮箱、角色、状态、创建时间

## 3. 技术实现

### 3.1 API 扩展

| API | 方法 | 功能 |
|-----|------|------|
| `/api/parties` | PATCH | 更新档案 |
| `/api/parties/[id]` | DELETE | 删除档案 |
| `/api/parties/export` | GET | 导出 Excel |
| `/api/admin/users` | PATCH | 更新用户角色/状态 |
| `/api/admin/users/[id]` | DELETE | 删除用户 |
| `/api/admin/users/export` | GET | 导出 Excel |

### 3.2 前端组件

| 组件 | 修改内容 |
|------|---------|
| `party-list.tsx` | 添加筛选、编辑弹窗、删除确认、导出按钮 |
| `user-list.tsx` | 添加筛选、滑动按钮、编辑弹窗、删除确认、导出按钮 |

---

## 4. 设计决策

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 状态切换方式 | 滑动即保存 | 操作简单，符合现代 UI |
| 档案编辑方式 | 弹窗编辑 | 清晰安全，符合数据管理规范 |
| 删除确认方式 | 弹窗确认 | 防止误删，安全第一 |
| 导出按钮位置 | 表格上方 | 最直观，用户习惯 |
| 删除用户限制 | 仅停用用户可删 | 安全保护，防止误操作 |
